Notes:

1) Uploaded images are cropped and resized to the default image size.
2) The FFT and convolved images display the amplitude values.
3) Enable “Sync Greyscale” to better visualize signal attenuation.
4) A collection of sample images is provided for educational purposes.
5) For the most accurate simulation of CTF effects, use a pixel size close to the actual imaging conditions. This may not be applicable or achievable for a random image.
Equations used:
1. φ = arctan(fy / fx)  
2. df = 0.5 · [ du + dv + (du – dv) · cos(2 · (φ – φa)) ]  
3. γ(f) = -(π / 2) · Cs · λ³ · f⁴ + π · df · λ · f² + p  
4. CTF(f) = –sin(γ(f) + arcsin(Ac)) · Etotal(f)  
5. Convolved_image = FFT⁻¹(FFT(Image) · CTF)
